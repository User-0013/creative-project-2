Anime/Manga Finder

Key Information

This applet allows you to search MyAnimeList for information on specific anime/manga. It also allows the searching of anime/manga characters. We only display the top fifty (50) results. Less results will be displayed when there aren't fifty (50) results. Click on the image to go directly to MyAnimeList to learn more about an anime/manga/character if it looks interesting!

This applet uses the Jikan API which limits the amount of bulk requests we can make to one (1) every four (4) seconds. Because of this, you can only submit one (1) search every four (4) seconds. We apologize for any inconvenience this may cause our users.
